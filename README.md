# 개발환경
- m1 MacPro
- node 18.12.0
- ruby 2.7.6
- react native 0.71
- iOS simulator: iPone 14(iOS 16.2)
- android simulator: Pixel 2_API_31(Android 12.0)
# 실행방법 
- npm install
- npx pod-install ios (pod 설치)
- npm run ios (ios 실행)
- npm run android (android 실행)
